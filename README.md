# Replit Clone

هذا المشروع يسمح للمستخدمين بكتابة أكوادهم وتشغيل بوتات Discord باستخدام التوكن الخاص بهم.

## كيفية التشغيل

1. قم بتثبيت الحزم:
   ```bash
   npm install
   ```

2. شغل الخادم:
   ```bash
   npm start
   ```

3. افتح المتصفح على الرابط:
   ```
   http://localhost:3000
   ```

4. أدخل الكود والتوكن واضغط على "Run".
